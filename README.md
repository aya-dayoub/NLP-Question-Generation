# NLP Question Generation System

This project was developed as my graduation project in Artificial Intelligence.

## Project Description
A Natural Language Processing (NLP) based system that generates exam-style questions from a given text.

## Features
- Text preprocessing and cleaning
- Tokenization and formatting
- Transformer-based model training
- Model evaluation and performance tuning

## Technologies Used
- Python
- NLP
- Transformer Models
- Data Preprocessing
